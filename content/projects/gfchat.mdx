gchat is a fullstack chat application. Front-end is built with Next.js and TailwindCSS. Back-end is built with Golang Gin framework and WebSockets.

### To run the application on your local machine:
start PostgreSQL database and create a database named `gchat`, then
```bash
cd .\server\
go run .\cmd\main.go
```
```bash
cd .\client\
npm install
npm run dev
```
